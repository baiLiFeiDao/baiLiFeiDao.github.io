标签的语义

标签的语义：标签的语法含义

标签效果：展示给用户看到的效果。通过样式控制，可以让标签达到相同的效果

标签的语义很重要：

1.有利于阅读，让代码的可读性变好

2.可以让网络搜索，检索到关键信息，并进行记录

3.语义化，可以让一些屏幕阅读器，针对这些标签有特殊操作，比如重音读，停顿等效果

4.在该语义下，会默认提供该语义化标签的效果

总结：使用一个标签，一定是因为在这个地方使用这种语义，而不是单纯的使用这种显示效果
